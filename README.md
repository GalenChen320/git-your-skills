## What is this?

`git-your-skills` is a meta-skill for version-controlling your skills. Now you can freely modify your skills however you want — no more worrying about breaking something with no way back.

## How to use

**Create a skill**
```
Create a skill called my-skill
Create a skill called my-skill, for summarizing my daily work report
```
*Note: If the frontmatter is modified, restart the opencode client for changes to take effect.*

**Initialize a skill**
```
Initialize my-skill
```
*Note 1: Skills created via `git-your-skills` are already initialized by default — no need to do it again. Note 2: If the frontmatter is modified, restart the opencode client for changes to take effect.*

**Update a skill**
```
Add a new feature to my-skill: auto-retry on error
```

**View history**
```
Show the last 10 changes to my-skill
```

**Compare two versions**
```
Compare the current version of my-skill with the last version from yesterday — what changed?
```

**Rollback**
```
Roll back my-skill to the version from last Monday
```

**Merge branches**
```
Merge the last version of my-skill from yesterday into the current version
```
