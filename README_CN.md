## 简介

`git-your-skills` 是一个元技能，用于对 skills 进行版本管理。现在你可以直接通过语言随意修改你的 skills，不用担心改崩后没法回滚了。


## 怎么用

**创建 skill**
```
帮我创建一个 skill，叫做 my-skill
帮我创建一个 skill，叫做 my-skill，用于处理每天的日报总结
```
*注意：如果修改了frontmatter，需要重启opencode客户端*

**初始 Skill**
```
初始化 my-skill
```
*注意 1：对于使用`git-your-skills`创建的skill，已经默认初始化，无需重复初始化。2：如果修改了frontmatter，需要重启opencode客户端*

**更新 Skill**
```
给 my-skill 加一个错误时能自动重试的新功能
```

**查看历史记录**
```
看看 my-skill 最近的10次修改
```

**对比两个版本**
```
对比 my-skill 现在的版本和昨天最后一个版本，修改了什么？
```

**回滚**
```
把 my-skill 回滚到上周一的那个版本
```

**合并分支**
```
把 sy-skill 的昨天最后一个版本合并到当前版本
```
